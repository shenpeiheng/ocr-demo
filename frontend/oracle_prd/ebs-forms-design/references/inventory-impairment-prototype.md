# 存货跌价计提原型参考

## 业务场景

季度存货计提及转销线上化处理。原手工 Excel → 系统线上化。

## 典型流程

```
库龄数据采集 → 跌价计算 → 转销计算 → 会计分录生成 → 传总账
```

## 功能模块划分（推荐6模块）

| 模块 | 核心内容 | 交互要点 |
|------|----------|----------|
| 计提比例配置 | 按存货类别（原材料/库存商品）× 库龄区间配置比例 | 可编辑表格+新增/编辑/保存 |
| 库龄数据采集 | 从 INV 模块取物料库龄数据 | 选择期间+存货类别→取数按钮 |
| 跌价计算 | 库龄金额 × 计提比例 = 计提金额 | 计算按钮+合计行+确认过账 |
| 转销计算 | 根据销售金额计算可转销金额 | 计算按钮+累计已转销+可转销金额 |
| 会计分录生成 | 自动生成跌价/转销凭证+传总账 | 生成预览+借贷平衡校验+传送 |
| 历史查询 | 各期计提/转销记录查看 | 期间范围+业务类型筛选+导出 |

## 典型库龄区间

| 区间 | 计提比例（参考） |
|------|----------------|
| 0 - 30 天 | 0% |
| 31 - 60 天 | 1% |
| 61 - 90 天 | 3% |
| 91 - 180 天 | 5% |
| 181 - 365 天 | 10% |
| 365 天以上 | 30% |

## 典型会计分录

### 跌价计提
- 借：6701 资产减值损失
- 贷：1471 存货跌价准备

### 转销
- 借：1471 存货跌价准备
- 贷：6701 资产减值损失

## C2 动作模式模拟数据要点

- **库龄采集**：8条物料（4原材料 + 4库存商品），含物料编码、名称、类别、数量、各区间金额
- **跌价计算**：逐条计算各区间 × 比例，汇总计提金额合计
- **转销计算**：本期可转销 = 按销售金额比例分摊，展示累计已转销
- **会计分录**：6条分录（4跌价 + 2转销），借贷平衡校验
