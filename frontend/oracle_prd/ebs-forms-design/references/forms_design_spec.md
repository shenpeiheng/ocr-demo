## 8. 表格配置

### 8.1 3D 雕刻边框（标准风格）

EBS 表格使用 3D 雕刻/浮雕边框效果，模拟传统 EBS Forms 的立体边框感。

**原理**：用亮/暗双色边框模拟立体感，上/左边框亮色（左上光源），下/右边框深色（阴影凹陷）。

## 10. 交互状态

| 状态 | 行为 |
|------|------|
| 默认 | 按钮浅蓝底 + 深蓝边 + 白色高光内阴影 |
| Hover | 背景变浅 (`#E6F2FF`)，无虚线轮廓 |
| Active | 背景加深 (`#C4D8E8`)，内阴影翻转 |
| Disabled | 灰底灰字，不可点击 |
| Focus (输入框) | 深蓝边框 + 蓝色外发光 |
| Selected (表格行) | 浅蓝背景，输入框白色+蓝边 |
| Balanced (合计) | 黑色差额文字 + 黑色边框标签 |
| Unbalanced (合计) | 黑色差额文字 + 红色边框标签 |

## 11. 多功能界面 — 左树右屏导航布局

当单个原型/系统需要承载多个功能模块且有层级关系时，使用 左侧树形导航 + 右侧独立功能面板 的布局。每个功能界面拥有独立的标题栏、菜单栏、内容区和状态栏。

### 11.1 整体布局

```
┌──────────────────────────────────────────────────────────┐
│ 标题栏（全局）                                            │
├──────┬───────────────────────────────────────────────────┤
│ 导航  │  ┌─ 功能面板标题栏 ───────────┬─ □ × ─┐        │
│ 栏    │  ├─ 功能面板菜单 ─────────────┤                │
│ 左侧  │  ├─ 功能面板内容区（滚动）──────┤                │
│ 固定  │  ├─ 功能面板状态栏 ───────────┤                │
│ 宽度  │  └──────────────────────────────────────────────┘ │
├──────┴───────────────────────────────────────────────────┤
│ 状态栏（全局）                                            │
└──────────────────────────────────────────────────────────┘
```

### 11.8 适用场景

- 单个原型承载 **7 个及以上**功能模块
- 功能模块有明确的父子层级关系（文件夹→叶子节点）
- 需模拟 EBS Forms 多表单切换操作体验
- 每个功能界面有独立查询条件、表格、操作按钮

### 11.9 CSS 类名约定

| 类名 | 用途 |
|------|------|
| `.tree-folder` | 文件夹容器 |
| `.tree-folder.open` | 展开状态，子节点可见 |
| `.tree-folder:not(.open)` | 折叠状态，子节点隐藏 |
| `.tree-leaf` | 叶子节点 |
| `.tree-leaf.active` | 叶子节点选中态 |
| `.tree-root-item` | 根级节点（总览等） |
| `.tree-root-item.active` | 根级节点选中态 |
| `.tab-pane` | 功能内容面板 |
| `.tab-pane.active` | 当前可见内容面板 |

### 11.10 JavaScript 参考实现

```javascript
// 切换功能面板
function switchPane(el, tabId, title) {
  // 1. 移除所有导航节点 .active，在 el 上添加 .active
  document.querySelectorAll('.tree-leaf, .tree-root-item')
    .forEach(n => n.classList.remove('active'));
  el.classList.add('active');
  // 2. 隐藏所有 .tab-pane，显示目标 tabId
  document.querySelectorAll('.tab-pane')
    .forEach(p => p.classList.remove('active'));
  document.getElementById(tabId).classList.add('active');
  // 3. 更新标题栏文字
  document.getElementById('viewTitleText').textContent = title;
}

// 展开/折叠文件夹
function toggleFolder(id) {
  document.getElementById(id).classList.toggle('open');
}
```
